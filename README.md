# Customer-Purchase-Analysis-Using-Python
This project aims to analyze e-commerce transaction data to gain insights into customer behavior, sales trends, and product popularity. The project also involves customer segmentation to identify different customer groups for business strategies.

# Dataset Information
•	Source: E-commerce transaction data
•	Columns:
o	CustomerID
o	Products
o	OrderDate
o	Spend
## Data Cleaning
•	Loaded the dataset using Pandas.
•	Handled missing values:
o	Dropped or imputed null CustomerID or Spend.
•	Formatted OrderDate to datetime format.
## Exploratory Data Analysis (EDA)
•	Analyze:
o	Total spend
o	Top customers
o	Popular products
•	Use Matplotlib and Seaborn for visualizations:
o	Sales trends
o	Customer patterns
## Customer Segmentation
•	Define customer types:
o	High Value
o	Regular
o	One-time Buyers

# Insights
•	Top 10% of customers contribute ~60% of total revenue.
•	Seasonal spikes in orders (analyze by month/quarter).
